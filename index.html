Frequently Asked Questions
And Installation Guide

Q: Will this mod work online with others and if so, will they see the Character and the Outfit? 


A: Yes, as long as they have the mod installed, if not they will see the default clothing and default character they picked.


Q: Do I need any DLC content in order to play as the new Character or use the weapons?
A: No, everything shown on this website is free and will always remain as free content.


Q: How do I Install this?
A: Simple, unzip everything from the archive to your PAYDAY 2 Root folder.
In other words, put all of the content from the archive to the folder where the .exe file is, since this mod is big in size and not just a simple "mod_overrides" type of mod. it uses both mods folder and the mod_overrides folder, so simply drop all the files to your PAYDAY 2 Root folder.


Q: Will I get banned if I play online with this pack installed?
A: No, not at all. You won't be banned by the game system or anything of that sort so don't worry about it





And that was for all the questions, thanks for checking in on the website. Below you will find buttons for mod dependencies that this mod uses by default. If you have those already, you can simply just install the mod pack and launch the game right away!
